# Guía de Instalación ## Requisitos previos - Microsoft Word (2010 o superior) - Navegador web moderno (Chrome, Firefox, Edge) - Cuenta en Canva ## Paso 1: Configurar la macro de Word 1. Descarga el archivo `WordToCanva.bas` desde la carpeta `src/word-macro/` 2. Abre Microsoft Word 3. Presiona `ALT + F11` para abrir el editor de Visual Basic 4. En el menú, ve a `Archivo > Importar archivo` 5. Selecciona el archivo `WordToCanva.bas` que descargaste 6. Cierra el editor de Visual Basic ## Paso 2: Configurar el bookmarklet de Canva ### Opción A: Arrastrar a marcadores 1. Visita la [página de demostración](index.html) de este repositorio 2. Arrastra el botón "WordToCanva Transfer" a tu barra de marcadores ### Opción B: Crear manualmente 1. En tu navegador, crea un nuevo marcador 2. Como nombre, pon "WordToCanva Transfer" 3. Como URL, copia el contenido del archivo `bookmarklet.js` 4. Guarda el marcador ## Paso 3: Usar la extensión 1. Abre tu documento en Word 2. Presiona `ALT + F8` para abrir el diálogo de macros 3. Selecciona "ExportToCanva" y haz clic en "Ejecutar" 4. Se guardará un archivo HTML en tu escritorio 5. Abre el archivo y copia todo el contenido 6. Ve a Canva y abre tu diseño 7. Haz clic en el bookmarklet "WordToCanva Transfer" 8. Pega el contenido cuando se te solicite ## Solución de problemas ### La macro no se ejecuta - Asegúrate de haber habilitado las macros en Word - Ve a `Archivo > Opciones > Centro de confianza > Configuración del Centro de confianza > Configuración de macros` - Selecciona "Habilitar todas las macros" ### El bookmarklet no funciona - Asegúrate de estar en la versión web de Canva (no la app de escritorio) - Intenta crear el bookmarklet manualmente ### El formato no se conserva completamente - La versión gratuita solo conserva formato básico - Considera adquirir la [licencia comercial](../docs/licencia-comercial.md) para funcionalidad completa
